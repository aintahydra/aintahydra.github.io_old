proofgeneral-on-osx

Installation (coq and proof-general)

$ brew install coq
$ brew install proof-general

Use

$ proofgeneral

Test

(When a “.v” file is read, it automatically changes to the “coq-mode”. Also, possible to change it manually by “M-x coq-mode”)

Open a new file “test.v” and type an example(from the book “Software Foundations”)

Now it’s ready to do interactive proof.

Keys (Keyboard shortcuts: “C-h m”):

  • C-c C-n : Next Step
  • C-c C-ENTER : Go to the point
  • C-c C-p : Display Proof State
  • C-c C-t : Display Context
  • C-c C-u : Undo Step
  • C-c C-r : Retract Buffer
